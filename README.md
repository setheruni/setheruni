## Featured Project

### 🎪 BounceBook | Party Rental Platform

A full-stack rental management system built from scratch to show how the pieces I work with every day fit together as a connected system. A real problem, a real data model, and a real API that two different front ends talk to simultaneously.

| Repo | What it is |
|------|-----------|
| [**bouncebook-api**](https://github.com/setheruni/bouncebook-api) | REST API built with Node.js, Express, and PostgreSQL |
| [**bouncebook-admin**](https://github.com/setheruni/bouncebook-admin) | Staff dashboard for booking management and business reporting |
| [**bouncebook-wordpress**](https://github.com/setheruni/bouncebook-wordpress) | Customer-facing WordPress booking site calling the live API |

The WordPress site checks real-time availability from the API. The admin dashboard approves or cancels those bookings through the same API. One database, one source of truth, two very different interfaces on top of it.
